# Testes automatizados feito no sistema feito em aula

Todos os comandos abaixo devem ser executados na pasta `backend`.

### Executar todos os testes

```bash
./mvwn test
```

### Para executar o End-to-End (E2E), rode o back e front end, depois o comando de teste

backend

``
./mvwn spring-boot:run
``

Frontend

``
npm run dev
``

### Comando para executar Teste

``
./mvwn -Dtest=NOMEDOARQUIVODETEST test
``
